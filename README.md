# Shopping Cart

A React shopping cart project from [The Odin Project React course](https://www.theodinproject.com/lessons/node-path-react-new-shopping-cart).

- Client-side routing with `Home`, `Shop`, and `Cart` pages
- Product data fetched from [FakeStore API](https://fakestoreapi.com)
- Shared cart state managed with React Context and `useReducer`
- Tests for reducer logic, page rendering, and component interactions

## Built With

- React
- Vite
- React Router
- Vitest
- React Testing Library

## Getting Started

```bash
npm install
npm run dev
```
